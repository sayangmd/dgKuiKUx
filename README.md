# 项目简介

基于深度学习的结核病检测系统，运用了YOLOv10算法和专门定制的YOLO数据集，旨在实现高效、准确的结核病检测。系统包括UI界面，便于用户操作，后端采用Python开发，提供了完整的源码和预训练模型。技术特点在于利用深度学习模型的强大识别能力，实现对医学影像中结核病征的自动识别与标注，助力医疗诊断工作。


## 核心功能
- 基于YOLOv10算法实现结核病的高效检测
- 使用专门定制的YOLO数据集进行模型训练
- 提供用户友好的UI界面，简化操作流程
- 完整的后端Python开发实现
- 包含完整的源代码和预训练模型
- 支持医学影像中结核病征的自动识别与标注

## 适用场景
- 医疗机构进行结核病的初步筛查
- 医学研究中的影像分析
- 医学教育中的案例展示
- 医疗辅助诊断工具

## 技术特点
- 采用深度学习技术进行医学影像分析
- 利用YOLOv10算法实现高精度目标检测
- 模型经过专业医学数据集训练
- 界面设计简洁直观，便于非专业人员操作
- 提供完整的开源代码，便于二次开发和学术研究
